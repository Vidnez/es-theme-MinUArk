**Optional (ArkOS Only)**

You can download in the "releases section" a loading screen and a boot logo. Installation instructions below.

<img width="1440" height="720" alt="preview3" src="https://github.com/user-attachments/assets/ee03a641-e4ba-4546-aac5-a7e657cfd03d" />


**Installation instructions:**

- For loading screen, copy loading.jpg to roms/launchimages or rooms2/launchimages

- Set "Game Loading Image Mode" to PIC and "Game Loading Image" as DEFAULT in Menu > UI Settings.

- For bootlogo, copy the "Image" file to the BOOT partition of your SD1 card overwriting the existing one.
